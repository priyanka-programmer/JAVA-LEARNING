# Java Learning Journey

This repository contains my Java practice programs and mini-projects as I learn core Java concepts. I'm building this step by step while working toward strong fundamentals in Java and problem-solving.

## What's inside

- **Calculator** : a console-based calculator supporting +, -, *, /, % with divide-by-zero handling
- **Armstrong** : checks if a number is an Armstrong number
- **Palindrome** :checks if a number/string is a palindrome
- **Fibonacci / Fiboseries** : generates Fibonacci series
- **Count / Counting / Five_count** : number counting practice programs
- **Sum_of_digit** : calculates sum of digits of a number
- **NestedSwitch / Switch** : practice with switch statements
- **Pattern** : pattern printing programs
- **largest** : finds the largest among given numbers

## Topics covered so far

- ✅ Variables & data types
- ✅ Loops (for, while)
- ✅ Conditionals (if-else, switch)
- ✅ Basic input handling (Scanner)
- ⬜ OOP (classes, inheritance, interfaces)
- ⬜ Collections
- ⬜ Data structures & algorithms
